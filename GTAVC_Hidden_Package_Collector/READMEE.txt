HOW TO USE:
1.. You need python 3 to be installed, inorder for the script to work
2.. Firstly, run requirement.bat file.(This File will  automatically get deleted after installing required modules..)
3.. then, run RUN_ME.bat
4.. don't press and hold the 'i' key, you might miss some packages.. possibly crash the game :)

NOTE : Don't press 'i' while being inside any building, try to be on streets...


Disclaimer::
The requirements.bat script will install some required python modules
